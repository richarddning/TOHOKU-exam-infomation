常见的拉普拉斯求导的结果：
| $f(t)$      | $F(s)$                        |
| ----------- | ----------------------------- |
| $1$         | $\frac{1}{s}$                 |
| $t$         | $\frac{1}{s^2}$               |
| $t^n$       | $\frac{n!}{s^{n+1}}$          |
| $e^{at}$    | $\frac{1}{s-a}$               |
| $t e^{at}$  | $\frac{1}{(s-a)^2}$           |
| $\sin(at)$  | $\frac{a}{s^2+a^2}$           |
| $\cos(at)$  | $\frac{s}{s^2+a^2}$           |
| $\sinh(at)$ | $\frac{a}{s^2-a^2}$           |
| $\cosh(at)$ | $\frac{s}{s^2-a^2}$           |
| $t\sin(at)$ | $\frac{2as}{(s^2+a^2)^2}$     |
| $t\cos(at)$ | $\frac{s^2-a^2}{(s^2+a^2)^2}$ |
